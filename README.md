# HyggeLang

HyggeLang er et fortolket, dansksproget prorammeringssprog,
bygget på principet om at kode skal være hyggeligt - ikke en pligt.

## Gundidé
HyggeLang er lavet for at udforske, hvordan programmering kan føles
mere hyggeligt, mere naturligt og lidt mere menneskeligt.

## Implementering
HyggeLang fortolkes i C#, så det bygges of køre direkte i .NET.
